# Returns{#returns}

vntc retourneert de volgende waarden, afhankelijk van het feit of een fout optreedt tijdens de verwerking.

* 0 als er geen fouten worden aangetroffen.
* 1 als er (fatale of niet-fatale) fouten optreden.
