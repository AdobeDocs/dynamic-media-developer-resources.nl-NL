## Eigenschappen {#section-5b75d459083f49f3bbd89ce8458559f7}

Tekstreeks. Vereist. Moet een geldig bestandspad voor afbeeldingsserver zijn, absoluut of relatief ten opzichte van `attribute::RootPath`.
