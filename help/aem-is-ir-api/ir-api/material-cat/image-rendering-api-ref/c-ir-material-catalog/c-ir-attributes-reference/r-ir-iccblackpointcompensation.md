# IccBlackPointCompensation{#iccblackpointcompensation}

Zwartpuntcompensatie. Geeft aan of zwartpuntcompensatie moet worden toegepast op kleurconversies wanneer er geen expliciete selectie is gemaakt met `icc=`.

## Eigenschappen {#section-21fd20b16bea4a22aecab0ae8b81e332}

Markering. Instellen op `0` om uit te schakelen of `1` compensatie van zwartpunten mogelijk te maken.

## Standaard {#section-5bc6703a43a149f18af88b70baae568f}

Overgenomen van `default::IccBlackPointCompensation` indien niet gedefinieerd of leeg.
