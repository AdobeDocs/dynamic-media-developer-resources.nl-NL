# TiffEncoding{#tiffencoding}

TIFF-coderingsindeling. Hiermee geeft u de compressie-indeling voor TIFF-afbeeldingen op (in feite de standaardinstelling voor de derde waarde van de `fmt=` ).

Instellen op `0` voor geen compressie, `1` voor LZW, `2` voor deflate (ZIP), en `3` voor JPEG-compressie.

## Eigenschappen {#section-469f5a1225464542866f5353edd92db3}

Enum.

## Standaard {#section-a3c5152a9f464e4987ed7c05d35b1169}

Overgenomen van `default::TiffEncoding` indien niet gedefinieerd of leeg.
