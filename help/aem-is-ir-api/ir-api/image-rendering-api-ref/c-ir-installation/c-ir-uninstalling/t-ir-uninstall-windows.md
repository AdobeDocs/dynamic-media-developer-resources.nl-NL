# Verwijderen in Windows{#uninstalling-on-windows}

Volg deze instructies om het teruggeven van het Beeld op een systeem van Vensters te desinstalleren.

1. Het verwijderprogramma starten door het uit te voeren ` *[!DNL install_folder]*\_uninst\uninstaller.exe`of in het menu Start selecteert u **[!UICONTROL Programs]** > **[!UICONTROL Scene7]** > **[!UICONTROL Image Rendering]** > **[!UICONTROL Uninstall]**.
