# Naam{#name}

Naam lettertype. De naam van het lettertype, exact zoals opgegeven in de opdracht RTF \fonttbl.

## Eigenschappen {#section-49d39cc6ec2b46c7a14e464dbe352f6a}

Tekstreeks. Vereist. Primaire indexsleutel voor de lettertypetoewijzing. Dezelfde font::Naam kan maximaal vier keer voorkomen in hetzelfde fonttoewijzingsbestand, maar elke record moet een andere combinatie van `font::Bold` en `font::Italic` waarden hebben.

## Standaard {#section-2bcf8ad7a7934956ad2cd42fd0d401f8}

Geen.
