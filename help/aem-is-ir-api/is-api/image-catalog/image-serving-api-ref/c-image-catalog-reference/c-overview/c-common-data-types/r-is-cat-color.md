# Kleur{#color}

Kleurgegevenstype.

Kleurwaarde. Hexadecimale, verpakte RGB-waarde, optioneel voorafgegaan door `0x`. De waarde RGB bijvoorbeeld `128,255,0` kan worden opgegeven als `0x80ff00` of `80ff00` .
