# Kleur{#color}

Kleurgegevenstype.

Kleurwaarde. Hexadecimale, verpakte RGB-waarde, optioneel voorafgegaan door `0x`. De RGB-waarde `128,255,0` kan bijvoorbeeld worden opgegeven als `0x80ff00` of `80ff00` .
