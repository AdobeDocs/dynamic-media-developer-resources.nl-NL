# UserData{#userdata}

Gebruikersgegevens. De server retourneert de inhoud van dit veld naar de client als reactie op req=userdata.

## Eigenschappen {#section-06f2002b77d54a64be07f12fff54ad13}

Tekstreeks. Het wordt aanbevolen de opmaak van [eigenschapsgegevens](/help/aem-is-ir-api/is-api/image-catalog/image-serving-api-ref/c-image-catalog-reference/c-overview/c-common-data-types/r-property-data.md) te gebruiken. Wanneer de gegevensopmaak van de eigenschap niet wordt gebruikt, mag de tekenreeks het teken &#39;=&#39; niet bevatten.

Dit veld neemt deel aan de lokalisatie van tekstreeksen. Zie Tekstreeks lokaliseren [in de](/help/aem-is-ir-api/is-api/http-ref/image-serving-api-ref/c-http-protocol-reference/c-syntax-and-features/r-text-string-localization.md) HTTP-protocolreferentie ** voor meer informatie.
