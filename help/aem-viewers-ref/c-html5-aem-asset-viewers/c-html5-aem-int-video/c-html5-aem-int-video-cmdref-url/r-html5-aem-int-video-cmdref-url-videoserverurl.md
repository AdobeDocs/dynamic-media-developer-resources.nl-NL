# videoServerUrl{#videoserverurl}

URL-opdracht voor video-viewer.

` videoServerUrl= *`videoRootPath`*`

  <tr> 
   <td colname="col1"> <p> <span class="codeph"> <span class="varname"> videoRootPath</span> </span> </p> </td>
