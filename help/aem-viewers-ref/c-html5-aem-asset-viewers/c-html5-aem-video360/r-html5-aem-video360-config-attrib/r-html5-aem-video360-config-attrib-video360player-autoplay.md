# Video360Player.autoplay{#video-player-autoplay}

Configuration attribute for Video360 Viewer.

`[Video360Player.|<containerId>_video360Player.]autoplay=0|1`

 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0|1  </span> </p> </td>

  </tr>
