# navigatie{#navigation}

URL-opdracht voor video-viewer.

` navigation= *`file`*`

De viewer ondersteunt de navigatie in videohoofdstukken door middel van gehoste WebVTT-bestanden. Operatoren voor activering worden niet ondersteund.
