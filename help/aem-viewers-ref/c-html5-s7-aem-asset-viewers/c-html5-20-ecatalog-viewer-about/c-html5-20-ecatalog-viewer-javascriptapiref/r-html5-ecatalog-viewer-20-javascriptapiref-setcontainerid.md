# setContainerId{#setcontainerid}

JavaScript API-referentie voor eCatalog Viewer.

` setContainerId( *`containerId`*)`

Hiermee wordt de id van de `DOM` container (gewoonlijk een `DIV`) waarin de viewer wordt ingevoegd. Het is niet nodig om het containerelement te hebben dat tegen de tijd wordt gecreeerd deze methode wordt geroepen. De container moet echter bestaan wanneer `init()` wordt uitgevoerd. Het moet eerder worden opgeroepen `init()`.

Deze methode is optioneel als de configuratiegegevens van de viewer worden doorgegeven met `config` JSON-object naar de constructor.
