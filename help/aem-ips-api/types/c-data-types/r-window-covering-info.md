| Naam | Type | Beschrijving |
|---|---|---|
| originalPath | `xsd:string` | Bestandspad. |
| originalFile | `xsd:string` | Bestandsnaam. |
| fileSize | `xsd:int` | Bestandsgrootte. |

| optimizedFile | `xsd:string` | Geoptimaliseerde bestandsnaam. |
