## Parameters {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

|---|---|---|
| name | `xsd:string` | Invoernaam. |

| compressedSize | `xsd:long` | Gecomprimeerde grootte. |
| uncompressedSize | `xsd:long` | Niet-gecomprimeerde grootte. |
