# getAssets{#getassets}

Keert activa van IPS terug.

Syntaxis

## Toegestane gebruikerstypen {#section-4673c1c9f4314160af8b165eb2dd20cc}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `TrialSiteAdmin`
* `TrialSiteUser`
* `ImagePortalAdmin`
* `ImagePortalUser`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>Retourneert alleen de elementen waartoe de gebruiker toegang heeft.
